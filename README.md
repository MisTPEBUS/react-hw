# Next.js 專案示範

**Demo 網站連結**：[react-hw-mocha Demo](https://react-hw-mocha.vercel.app/)

這是一個使用 [Next.js](https://nextjs.org) 建立的專案，並透過 [`create-next-app`](https://nextjs.org/docs/app/api-reference/cli/create-next-app) 快速啟動。

---

## 快速開始

首先，安裝專案所需的套件：

```bash
npm install
# 或是
yarn install
# 或是
pnpm install
# 或是
bun install
```
